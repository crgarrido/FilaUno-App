# Fila Uno

Sitio publico de Fila Uno para App Store, Google Play, soporte y politica de privacidad.

## Publicar con GitHub Pages

1. Sube estos archivos al repositorio `crgarrido/FilaUno-App`.
2. En GitHub entra a `Settings > Pages`.
3. En `Build and deployment`, selecciona `Deploy from a branch`.
4. Elige la rama `main` y la carpeta `/root`.
5. Guarda.

La URL quedara parecida a:

```txt
https://crgarrido.github.io/FilaUno-App/
```
